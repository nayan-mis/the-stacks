connect this site to GitHub

Publishing works by committing files to the same GitHub repo this site is hosted from. 1) Create a fine-grained personal access token at github.com/settings/personal-access-tokens/new scoped to only this one repo, with Contents: Read and write permission — nothing else. 2) Paste the repo details below. The token is saved only in this browser's local storage — never committed to the repo, never sent anywhere but GitHub's API.
